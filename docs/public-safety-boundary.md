# Public Safety Boundary

This document defines what Anvil intentionally excludes to maintain a clean separation from any proprietary work.

## What This Repo Contains

- A clean-room implementation of multi-model adversarial code verification
- Synthetic examples and toy services for demonstration
- Generic prompt templates and guardrails
- Public architecture documentation refined through open adversarial review
- All code written from scratch for this public project

## What This Repo Excludes

This repository contains **none** of the following:

- **Proprietary code** — no code from any employer, past or present
- **Internal documents** — no internal design docs, runbooks, or wiki content
- **Internal prompts** — no production prompt templates or agent configurations
- **Internal project names** — no real project names, team names, or product names
- **Internal source paths** — no real file paths, repo names, or directory structures
- **Proprietary examples** — no real task descriptions, claims, issues, or run artifacts
- **Real logs or artifacts** — no production run logs, scorecards, or event traces
- **Unreleased product references** — no references to internal or unreleased tools
- **Personal data** — no names, emails, or identifiers of colleagues or collaborators

## Origin Statement

Anvil is a clean-room public implementation inspired by patterns observed while using AI coding agents in production engineering workflows. The architecture, code, documentation, examples, and prompts in this repository were created independently for public release.

## Example Data

All examples in this repository use synthetic data:

- `examples/toy-python-service/` — a minimal fake Python service for demonstration
- `examples/sample-run-artifacts/` — synthetic run artifacts showing the pipeline output format
- All file paths, function names, variable names, and test cases are fictional

## Prompt Templates

Prompt templates in this repository are generic and written specifically for the public version. They do not reproduce, paraphrase, or derive from any internal prompt engineering work.

## If You Have Questions

If you believe any content in this repository inadvertently includes proprietary material, please open an issue and it will be addressed immediately.
